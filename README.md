# Prince Edward School Rugby - The Tigers

A modern, responsive React application showcasing the rich history and legacy of Prince Edward School Rugby.

## Overview

This website celebrates over 125 years of rugby tradition at Prince Edward School in Harare, Zimbabwe. It features the school's rugby identity, historical timeline, notable alumni, technical team, and enduring legacy.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, elegant design using the school's colors (Maroon, Green, White)
- **Smooth Animations**: Scroll-triggered animations and smooth transitions
- **Interactive Timeline**: Visual timeline showcasing key moments in the school's rugby history
- **Navigation**: Fixed navigation bar with smooth scrolling
- **React Components**: Modular component architecture for easy maintenance
- **Sections**:
  - About Prince Edward School
  - Rugby Identity & Culture
  - History & Traditions
  - Historical Timeline
  - Notable Alumni
  - Technical Team (Coaching Staff)
  - Enduring Legacy

## Technologies

- **React 18**: Modern React with hooks
- **CSS3**: Custom styling with CSS Grid and Flexbox
- **Create React App**: Build tooling and development server
- **Google Fonts**: Inter & Playfair Display

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
